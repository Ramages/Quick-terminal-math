# Quick-terminal-math
An easy way to execute mathematical operations from the terminal

Supported operations<br/>
#### add
```
math 5 + sqrt6

>> 7.449490
``` 
#### sub
```
math 5 - sqrt6

>> 2.550510
```
#### divide
```
math 5 / sqrt6

>> 2.041241
```

#### multiply
```
math 5 x sqrt6

>> 12.247450
```

#### raise
```
math 5 ^ sqrt6

>> 2.236068
```

#### modular
```
math 5 % sqrt6

>> 1
```
#### sqrt
```
math sqrt36

>> 6
```
